⚙️ 設定 Google Sheets 自動記錄

完成以下步驟，巡查記錄將自動儲存至 Google Sheets 並發送電郵至 saiwanhsrecord@gmail.com

1

開啟 Google Apps Script

前往 script.google.com，以您的 Google 帳號登入，建立新專案。

2

貼入附帶的 .gs 腳本

將下載包中的 inspection-gas-script.gs 內容全部貼入 Apps Script 編輯器，然後儲存。

3

連結 Google Sheets

在 Apps Script 中，點擊左側「服務」→ 加入 Google Sheets API 及 Gmail API。然後在右上角選擇連結到一個新的或現有的 Google Spreadsheet。

4

部署為 Web App

點擊右上角「部署」→「新增部署」→ 類型選 網頁應用程式。
執行身分：我｜存取權：任何人。完成後複製 Web App 網址。

5

貼入 Web App 網址

將複製的網址貼入下方，儲存後即可自動傳送記錄。
